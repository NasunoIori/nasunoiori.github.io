Mesh 组网来了 —— 90 平米户型体验 TP-LINK 一键互联

发表于 2020-03-15   |   分类于 Computer and Internet

“我这个房间网速就是这么慢，打开百度都要等 2 分钟” —— imouto

最近因为疫情的原因一直在家休息，网络也就成了和外界仅剩的联系方式，前两天我妹跑过来说她房间网速太差，光是打开百度搜索都得要 2 分钟，还特意给我演示了一番，我拿我手机到她房间测试也是如此，于是具体分析了一下房间网络架构后，决定购入新的路由器，进行 mesh 组网，全屋漫游。

什么是 Mesh 组网

当家里出现 Wi-Fi 信号覆盖不全的时候，我首先想到的就是桥连路由器，但是实际上看了网上大家的分享结合以前自己的实际体验后发现，仅仅用多个路由器扩展 Wi-Fi 信号覆盖范围存在自动切换不及时需要手动切换路由的情况，而且切换过程等待时间长，对于在线视频影响还算小，但是对于即时网络聊天等等就是噩梦一般的体验了。

这个时候就需要 Mesh 组网技术啦，那么什么是 Mesh 网络？TP-LINK 和 Linksys 都在官网中介绍了这个名叫“网状网络”的技术：把原有的单个路由想象为房间里的一个喇叭，在隔壁房间当然是声音较小甚至听不到，如果我们在每个房间都放上一个喇叭并和主喇叭连起来，这样每个房间都能听见喇叭的声音，并且从一个房间走去另外一个也不会有障碍。

TP-WDR7651



从拼多多买的 TP-WDR7651，这包装盒磕碰得像是二手货，好在内容物没有问题，背面着重介绍了 TP-LINK 自家“易展”技术，也就是 Mesh 组网技术。

联机测试

首先检查了一下家里光猫的设置，原来的网线是五类线，最高只支持 100M 带宽，正好买路由器送了超五类线，二话不说先换上，虽然现在家里只有 100M 宽带，但是说不准什么时候就换了呢。

按照说明书连上路由器电源，插上网线，等待路由器开机好，设置完管理员密码、上网宽带帐号密码后就可以连上互联网啦。顺带把说明书和便笺贴放在路由器下，方便我去学校时爸妈也能操作。

接着把7650也拿出来，插上电，先后依次按下“易展”按钮，不出一分钟就设置好了，真的是很方便。挪去我妹房间后也是很快连上。

网络设置

连上后用电脑登录网关，看看 Mesh 组网下的网络拓扑结构。可以看见主路由 7651 下挂载了 7650 分路由，另外还有支持 HomeKit 的智能灯泡、我的 iPhone、电脑、Apple Watch 和我的另一台电脑。而 7650 路由器下则挂载着在厨房做菜的爸妈的手机、我妈的 iPad 和我妹的 iPhone。我实测后没有发现存在网络切换卡顿之类的问题，全家人体验一天后也是表示厨房和我妹房间网络状况大有改善，体验甚佳。

另外还有一点值得提的就是这个 Wi-Fi 多频合一的功能，原来我的智能灯泡和 Apple Watch 只支持 2.4 GHz Wi-Fi，只要手机连上了 5 GHz Wi-Fi 就存在不能控制灯泡的问题，现在算是彻底解决了这个痛点。

总结

Pros：

  • 旧式 90 平米房屋虽然面积不大，但是在承重墙较多的条件下也可以做到有效覆盖
  • “易展”功能真的很好用，适合不会设置路由器的多数群体
  • 超五类线好评

Cons：

  • 下单完第二天 TP-LINK 就推出了带“易展”功能的 Wi-Fi 6 路由器，擦汗
  • 价格虽然已经很便宜了，但是多个路由器加起来还是一笔不小的开支

以上就是本次使用的全部体验了，147+145 元的价格稍稍有点贵，但是也可以接受。
