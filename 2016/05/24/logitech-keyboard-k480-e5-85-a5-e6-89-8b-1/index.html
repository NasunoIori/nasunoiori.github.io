Logitech Keyboard K480入手

发表于 2016-05-24   |   分类于 Computer and Internet ， Sharing

前几天在闲鱼上以低价收购了一台蓝牙键盘，Logitech Keyboard K480，除去邮费100软妹币，使用了几天，感觉还不错，分享一下

1. 外观

[caption id=”” align=”alignnone” width=”3264”] k480 正脸照[/caption]

选择的是黑色加柠檬黄（好像是叫这个？），和我自己手上的寨板还挺配的，但是由于是二手，所以成色当然是一般般了，不过也没有什么大的瑕疵

[caption id=”” align=”alignnone” width=”3264”] K480 按键细节[/caption]

罗技不愧是做键盘的行家，虽然这款K480算是低端（根据价格的臆测）产品，但是还是有很多细节方面的考虑，按键最下面一行都是凸起的手感（最上面一行的功能键也是，但是这张照片里面没有拍到），而中间字母和数字则是凹进去的手感，更加贴合指腹，算是人体工学吧。另外，所有按键都是磨砂质感而键盘表面则是类肤材料，打字时候也不容易出错。

2. 功能

2.1 蓝牙兼容性————中等偏上

作为一款蓝牙键盘，首先要说的当然还是蓝牙功能，兼容性方面，K480可以配对3个不同的蓝牙设备，根据官方介绍包括Windows，OS X，iOS，Android平台，甚至还有Chrome OS平台（当然，WP是不支持的：实测lumia 920可识别，可连接，但是打不了字……），我自己手边的iPhone6和苦逼魔方Bug10平台的寨板经测试可以顺利连接上，但是5年前买的笔记本就连不上了（可能是蓝牙2.0的锅）。

按照罗技官方说明书上的说明，对应不同系统有着不同的功能快捷键，从我这几天使用的结果看来，都还不错，iOS平台的home键，多任务界面呼出，虚拟键盘呼出，音量调节，音乐播放/暂停/上一首/下一首等等都可以正常使用，Win平台也是如此，可以认为兼容性还算不错。

2.2 蓝牙配对难易度————易

[caption id=”” align=”alignnone” width=”3264”] K480 蓝牙切换旋钮[/caption]

K480的初始蓝牙配对连接是属于傻瓜类型的，按住特定的连接按键不放直到指示灯闪烁，用设备搜索并连接键盘，可能有的设备要输入配对码，输入完回车就OK了，再次连接什么都不用管可以自动连接，实在是很方便，这点还是要点赞的。

不同设备之间可以使用旋钮切换，切换之间大概有几秒钟的延时，在可接受范围内。

2.3 蓝牙稳定性————一般

我现在就是在用这款键盘写blog，稳定性方面只能说是一分价钱一分货。买来的一周时间之内，有一天晚上突然就没法用了，虽然果6和寨板都能识别并连接上，但是按键就是没有反应，最后还是抠电池+关机重开解决问题，除此之外，其余时间内稳定性都比较高。可能我现在处理文书方面的工作也在不断增加，所以对稳定性这方面的要求比较高，因此，稳定性这方面只能勉强算个及格，毕竟不是每个老板都有心情等着你给什么牙键盘修什么bug的。

3. 其他

3.1 电池

[caption id=”” align=”alignnone” width=”3264”] K480 电池仓[/caption]

K480使用两节7号电池（AAA电池）供电，开机键在电池仓上，有一个LED灯在开机时作为指示。续航的话还没测试过，二手买来时候的电池用了一个星期还有电（每天大约4~5小时使用时长）。

3.2 设备支架

[caption id=”” align=”alignnone” width=”3264”] 支撑寨板的形态[/caption]

K480的键盘布局前方有一个带角度的凹槽，用来放所连接的设备，我的10.1寸寨板将将好放进去，所以比这个尺寸大的设备就不能愉快地使用这个方便的功能了。支撑起来之后角度虽然是固定的，但是还算合适，看看电影，打打游戏什么的也完全不成问题。

3.3 自重及防滑脚垫

说到重量，用一句话概括就是：不适合携带。为了发挥上面说到的设备支架的功能，键盘的后方应该是添加了配重以防止设备加上后重心不稳而后倾，所以整个设备显得比较沉而且重心不稳，但是这样扎实的底座也让防滑脚垫发挥了作用，打字过程中不会因为速度过快而移动整个键盘。

3.4 软件配套

[caption id=”” align=”alignnone” width=”1548”] Logitech Options软件界面1[/caption]

罗技官方也为K480等一系列蓝牙键盘配了Logitech Options软件以辅助其发挥更多的功能，包括设置功能键和多媒体键之间的切换，大写锁定按键和电池电量提示，比较方便。

[caption id=”” align=”alignnone” width=”1548”] Logitech Options软件界面2[/caption]

4. 总结

综上所述，感觉这款蓝牙键盘符合自己的心理预期，价钱也是相当合理，不是买了之后就后悔的电子产品。有了键盘之后寨板的工作效率直线上升，果然Win系统还是要配键盘和鼠标。

PS：闲鱼上还是有不少好东西的，下一次有空就分享更早之前在闲鱼上收的二手wacom绘图板吧。
